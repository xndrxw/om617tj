# OM617 Specifications

## General

| Item | Specification |
|---|---|
| Engine | Mercedes OM617 |
| Configuration | Inline 5-cylinder diesel |
| Aspiration | Turbocharged |
| Fuel system | Mechanical injection |
| Oil filter used on this build | Baldwin B2-HPG |

## Build-Specific Baseline

- Turbocharger: Holset HE221W, 7cm housing, internal wastegate
- Boost target: approximately 30 PSI
- Injection pump: 7.5mm currently installed
- Planned pump upgrade: 8mm M-pump
- Injectors: upgraded Bosch units
- Intercooling: water-to-air with FrozenBoost core and Meziere WP136S pump
- Installed chassis: 2005 Jeep Wrangler TJ

## Rebuild / Service Notes Currently Backed by Wiki Content

The local OM617 wiki has a detailed rebuild torque reference pulled from the OM617 manual. High-value values already captured there include:

- Main bearing bolts: 90 Nm
- Connecting rod nuts: 40-50 Nm + 90-100°
- Crank bolt M18 x 1.5 x 45: 270-330 Nm
- Flywheel / driven plate bolts: 30-40 Nm + 90-100°
- Hex head bolts: 70 -> 90 -> wait 10 min -> 100 Nm
- Double-hex necked head bolts: 40 -> 70 -> wait 10 min -> 90° -> 90°
- Cam sprocket bolt: 80 Nm
- Chain tensioner closing plug: 90 Nm
- Prechamber threaded ring: 150-180 Nm
- Injection nozzles / nozzle holders in prechamber: 70-80 Nm
- Glow plugs: 20-30 Nm

## Documentation Boundary

This repo page is the quick build-facing summary only. The deeper torque-sheet and OCR-sensitive notes live in the local OM617 wiki and manual-derived references, which remain the better source for engine assembly work.
