# Fuel System Overview

## System Type

Dual fuel system using separate tanks for diesel and waste oil.

## Tanks

- Driver side: diesel
- Passenger side: waste oil
- Both tanks use Beans Diesel sump fittings
- -10AN lines from tanks

## Filtration

- Primary filters: Baldwin BF1272-SPS, one per tank
- Final filter: Donaldson P551313
- Engine oil filter: Baldwin B2-HPG
