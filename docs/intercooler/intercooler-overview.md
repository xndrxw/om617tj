# Intercooler System Overview

## System Type

Water-to-air intercooler system designed around the packaging realities of the TJ chassis and the heat load of a boosted OM617.

## Current Components

- FrozenBoost Water to Air Intercooler - 11"x9.25"x3.5" - Bend I/O (Type 13)
- Installed core reference URL: https://www.frozenboost.com/air_water-ic/water-to-air-intercoolers-p-213.html
- Air to Water Intercooler Aluminum Heat Exchanger Radiator universal 31" x9" x 2.25" + Fans
- Installed heat exchanger reference URL: https://www.amazon.com/dp/B085Y56RWP?ref_=ppx_hzsearch_conn_dt_b_fed_asin_title_17
- Meziere WP136S circulation pump
- Intercooler plumbing with silicone hose
- Planned / desired reservoir addition
- Dual 8-inch fan upgrade path already noted in the wiki

## Sensor Coverage

- Intercooler coolant temperature
- Intake air temperature before the intercooler
- Intake air temperature after the intercooler

## Design Goals

- Hold charge-air temperatures down under sustained load
- Make the system bleedable and serviceable
- Keep plumbing manageable in the TJ layout
- Avoid pump starvation, trapped air, and rear heat-exchanger airflow problems

## Known Watch Areas

- Rear heat-exchanger airflow effectiveness at low vehicle speed
- Pump placement and priming behavior
- Reservoir sizing and bleed strategy
- Hose routing, abrasion, and trapped-air points
- Total heat-rejection margin once boost and fueling are both pushed harder

## Current Upgrade Path Already Identified

- Improve rear fan package
- Add intercooler reservoir volume
- Revisit water/meth only if charge-air control still needs help after the core system is optimized
