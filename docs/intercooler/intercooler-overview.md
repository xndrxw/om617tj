# Intercooler System Overview

## System Type

Water-to-air intercooler system.

## Components

- FrozenBoost intercooler
- Rear-mounted heat exchanger
- Meziere WP136S pump
- Silicone hose
- Planned reservoir

## Sensor Locations

- Intercooler coolant temperature
- Intake air temperature pre-intercooler
