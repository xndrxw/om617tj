# Power Distribution

## Main Components

- Dual AGM batteries
- Blue Sea Systems battery selector
- 500A breakers
- ANL protection for major loads
- Custom fuse / relay box
- Premier Power Welder
- AD244 alternator

## Major Loads Currently Called Out

- Starter
- Glow plugs
- Winches
- Radiator fan
- Intercooler pump
- Rear heat-exchanger fan package
- Lighting
- Air compressor
- Radios
- Gauges and sensor system

## System Priorities

1. Keep high-current paths short, obvious, and protected.
2. Make branch-circuit protection easy to inspect in the field.
3. Separate dirty power loads from instrumentation and sensor wiring.
4. Preserve manual understanding of the system instead of burying function in unnecessary relay complexity.

## Known Power-Distribution Risks

- Voltage sag during simultaneous high-current loads
- Incomplete fuse / breaker schedule documentation
- Cable-size mismatch to real load and duty cycle
- Poor physical separation between power and sensor harnesses
- Service difficulty if distribution hardware is not clearly labeled

## Documentation Gap That Still Matters

The hardware baseline is known, but the repo still needs exact protection values, cable gauges, and branch-by-branch mapping before this page becomes a true wiring reference.
