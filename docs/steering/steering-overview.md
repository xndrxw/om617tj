# Steering System Overview

## System

- Hydro assist steering
- Stock TJ steering box tapped for assist
- PSC ram
- Shared reservoir / filter setup
- Power steering cooler

## Design Intent

The steering system is built to handle large tires and off-road steering loads without jumping to a fully custom steering-box conversion.

## Real Engineering Concerns

- Hose routing around heat, abrasion, and suspension travel
- Fitting selection and pressure-rating margin
- Ram mount and steering-link load paths
- Cooler sizing and airflow under slow technical use
- Contamination control in the shared reservoir / filter loop
- Steering feel and return-to-center after assist tuning
