# OM617TJ

> A fully custom OM617-powered Jeep Wrangler TJ built for expedition travel, rock crawling, fabrication, thermal efficiency, and mechanical simplicity.

The OM617TJ is a heavily modified 2005 Jeep Wrangler TJ powered by a Mercedes-Benz OM617 turbo diesel engine. This repository serves as the primary engineering documentation archive, technical wiki, fabrication log, and long-term build record for the project.

The vehicle combines:
- Mechanical diesel simplicity
- Heavy-duty drivetrain components
- Extensive thermal management systems
- Fully custom electrical integration
- Long-travel suspension
- Expedition-focused reliability
- Extreme low-range crawl capability

This repository exists to preserve technical knowledge, fabrication details, service procedures, component references, and future development plans related to the build.

# Build Philosophy

The OM617TJ was designed around several core principles:

- Mechanical simplicity
- Long-term reliability
- Field serviceability
- Thermal efficiency
- Heavy-duty drivetrain strength
- Modular electrical systems
- Off-road capability
- Redundancy where possible
- Minimal dependency on factory electronics

The project intentionally avoids unnecessary electronic control systems while integrating modern monitoring, cooling, charging, and fabrication techniques.

# Major Systems

# Engine System

The engine system is based around a heavily modified OM617 turbo diesel platform featuring:

- Ported cylinder head
- Holset HE221W 7cm turbocharger
- Port-matched turbo, header, and downpipe
- Jet-Hot ceramic coated hot side components
- 7.5mm BenzForce injection pump
- Benz Injection injectors
- Air-to-water intercooler system
- Water-methanol injection
- Custom ported naturally aspirated intake manifold
- Cometic MLS head gasket
- P1 Manufacturing head studs

Future plans include:
- Forged connecting rods
- Total Seal piston rings
- P1 Manufacturing main studs
